What programming languages are taught in the Computer Science program?

12 answers • 45 upvotes
Tendai Moyo Computer Science Expert 450 pts
Answered 1 day ago

The Computer Science program at NUST covers several programming languages:

  1. Python - Used in first year for fundamentals and data structures
  2. Java - Object-oriented programming in second year
  3. C++ - Systems programming and algorithms in third year
  4. Web Technologies - HTML, CSS, JavaScript for web development

The curriculum is designed to build from simple to complex concepts, preparing students for real-world software development.
